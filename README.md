# nuxtjs-template
Nuxt.js, SCSS, Pug, PWA  

## Installation

    $ git clone https://github.com/115design/nuxtjs-template.git
    $ cd nuxtjs-template
    $ yarn install
    $ yarn dev
      or
    $ yarn start

Nuxt.js: http://localhost:3000
